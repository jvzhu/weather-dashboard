# weather-dashboard

A responsive weather dashboard built with React, TypeScript, Tailwind CSS, and the Open-Meteo API.

## Features

- Current weather details with humidity, wind speed, UV index, pressure, visibility, and precipitation
- 24-hour scrolling forecast and 5-day outlook
- City search with autocomplete suggestions
- Browser geolocation on load with graceful fallback
- Persisted Celsius/Fahrenheit and dark/light theme toggles
- Weather alert panel and resilient loading/error states

## Development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```
